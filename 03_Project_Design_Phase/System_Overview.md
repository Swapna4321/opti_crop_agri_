# System Overview

OptiCrop is an AI-powered crop recommendation system that assists farmers in choosing the best crop for cultivation.

## Inputs

- Nitrogen (N)
- Phosphorus (P)
- Potassium (K)
- Temperature
- Humidity
- pH Value
- Rainfall

## Processing

- Data Collection
- Data Validation
- Machine Learning Prediction

## Output

- Recommended Crop
- Decision Support for Farmers

## Users

- Farmers
- Agricultural Researchers
- Students
