# User Stories

## Farmer User Stories

### Story 1
As a farmer, I want to enter soil nutrient values (Nitrogen, Phosphorus, and Potassium) so that I can get the most suitable crop recommendation.

### Story 2
As a farmer, I want to provide environmental parameters such as temperature, humidity, pH, and rainfall so that the system can analyze field conditions accurately.

### Story 3
As a farmer, I want the system to recommend the best crop for cultivation so that I can improve productivity and profitability.

### Story 4
As a farmer, I want a simple and easy-to-use interface so that I can use the application without technical knowledge.

### Story 5
As a farmer, I want reliable recommendations so that I can reduce the risk of crop failure.

## Administrator User Stories

### Story 6
As an administrator, I want to manage agricultural datasets so that the prediction model remains accurate and updated.

### Story 7
As an administrator, I want to monitor system performance so that the application works efficiently.
